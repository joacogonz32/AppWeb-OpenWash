# AppWeb-OpenWash

**OpenWash** es una aplicación web diseñada para la gestión y administración de un servicio de lavandería. Permite a los usuarios registrar, consultar y hacer seguimiento de pedidos de lavado, facilitando tanto la experiencia del cliente como la operación interna del negocio.

## Características principales

- Gestión de pedidos de lavandería
- Seguimiento del estado de los servicios
- Administración de clientes
- Interfaz web intuitiva y accesible
